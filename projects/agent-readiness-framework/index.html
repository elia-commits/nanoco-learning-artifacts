Core Principle

כל עובד הופך למנהל סוכן

Every employee becomes an Agent Manager

כל עובד מבין מה הסוכן שלו עושה, מנחה אותו, ואחראי על התוצרים שלו.

Each employee understands what their agent does, directs it, and is accountable for its outputs.

מה הוא עושה What it does להכיר את היכולות האמיתיות של הסוכן. Know the agent's actual capabilities.
איך הוא משתלב How it fits שימוש בתוך העבודה היומיומית, לא לצידה. Use it inside daily work, not alongside it.
מי מנחה אותו Who directs it העובד נשאר בשליטה. The employee stays in control.
מה מותר What's allowed כללים ברורים לגישה ולאישורים. Clear rules on access and approvals.
What It Takes

מה נדרש כדי שזה יעבוד

What It Takes To Make It Work

סוכנים לא מייצרים ערך לבדם. האנשים שמשתמשים בהם, האופן שבו הם מוגדרים, והכללים סביבם — כולם משמעותיים.

Agents don't deliver value on their own. The people using them, the way they're set up, and the rules around them all matter.

01 להכיר מה סוכנים יכולים. Know what agents can do.
02 להשתמש בהם באחריות. Use them responsibly.
03 להגדיר אותם לתפקיד. Configure them for the job.
04 להגדיר מה מותר. Define what's allowed.
